# 💊 Já Tomei?!

> Uma forma elegante e simples de nunca mais esquecer sua saúde.

O **Já Tomei?!** é um aplicativo multiplataforma desenvolvido em Flutter, focado na gestão de medicamentos com uma experiência de usuário (UX) premium. O objetivo é transformar a tarefa de tomar remédios em algo intuitivo, através de um design sereno e funcionalidade inteligente.

---

## ✨ Funcionalidades (Roadmap)

- [x] **Dashboard Inteligente:** Resumo diário do progresso e próximo medicamento em destaque.
- [ ] **Lista de Medicamentos:** Gerenciamento completo (cadastro, edição e estoque).
- [ ] **Histórico Visual:** Calendário detalhado para acompanhar a evolução e adesão ao tratamento.
- [ ] **Notificações Críticas:** Alertas personalizados para garantir que nenhum horário seja perdido.

## 🎨 Design System

O app utiliza uma paleta de cores baseada no **Azul Sereno**, focada em transmitir calma e confiança. 
- **Estilo:** Material 3 com toques de Glassmorfismo e Neomorfismo.
- **Tipografia:** Moderna e legível, com hierarquia visual clara.
